## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript api repository for vetverse application.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
